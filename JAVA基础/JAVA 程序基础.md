# JAVA 程序基础

### 2.3.2 局部变量规则
- Java 局部变量（方法内定义的变量）**必须显式初始化**，不能直接使用未赋值的局部变量进行运算

### 2.3.3 整数运算特性
- 整数运算的结果永远是精确的；
- 运算结果会自动提升数据类型；
- 可以强制转型，但**超出范围的强制转型会得到错误结果**；
- 应选择合适范围的整型（int 或 long），无需为节省内存使用 byte/short 进行整数运算

### 2.4 字符串与分支语句
#### 字符串比较
- 字符串内容**禁止使用 `==` 比较**（需使用 `equals()` 方法）

#### switch 语句规则
- switch 语句可实现多重选择，执行匹配 case 后的代码；
- switch 计算结果类型限制：**整型、字符串或枚举类型**；
- ⚠️ 注意：千万不要漏写 `break`，建议开启 fall-through 警告；
- ✅ 规范：总是写上 `default` 分支，建议开启 missing default 警告；
- 语法升级：从 Java 14 开始，switch 正式升级为表达式，无需 break，可通过 `yield` 返回值
